#### 2. اضافه کردن Fuzzy Matching
```python
from fuzzywuzzy import fuzz

def find_similar_entities(query_entity: str, db_entities: List[str], threshold=80):
    """یافتن نزدیک‌ترین entity"""
    matches = []
    for entity in db_entities:
        score = fuzz.ratio(query_entity, entity)
        if score >= threshold:
            matches.append((entity, score))
    
    return sorted(matches, key=lambda x: x[1], reverse=True)
```
