# فولدر Archive - فایل‌های اضافی پروژه

این فولدر شامل تمام فایل‌های اضافی پروژه است که برای نگهداری و مراجعه آینده نگه داشته شده‌اند.

## ساختار فولدرها

### 📁 tests/
شامل تمام فایل‌های تست (`test_*.py`) که برای تست و دیباگ سیستم استفاده می‌شدند.

### 📁 logs/
شامل تمام فایل‌های لاگ (`.log`) که برای ردیابی و دیباگ تولید شده‌اند.

### 📁 reports/
شامل تمام گزارش‌های تحلیل و مستندات (`.md`) که در طول توسعه پروژه نوشته شده‌اند.

### 📁 json_files/
شامل فایل‌های JSON که برای تست و دیباگ استفاده شده‌اند.

### 📁 scripts/
شامل اسکریپت‌های کمکی مانند:
- `fix_*.py` - اسکریپت‌های رفع مشکل
- `process_*.py` - اسکریپت‌های پردازش
- `debug_*.py` - اسکریپت‌های دیباگ
- `analyze_*.py` - اسکریپت‌های تحلیل

### 📁 data_files/
شامل فایل‌های داده نمونه (Excel, PDF) که برای تست استفاده شده‌اند.

---

**توجه:** این فایل‌ها برای مراجعه و نگهداری نگه داشته شده‌اند و برای اجرای سیستم ضروری نیستند.
